# Моди

Hytale Server підтримує моди для розширення геймплею.

## Встановлення модів

### Спосіб 1: Файловий менеджер (Рекомендовано)

1. Відкрийте веб-панель
2. Перейдіть на вкладку **Mods**
3. Перегляньте доступні моди з Modtale
4. Натисніть **Install** на моді, який хочете

### Спосіб 2: Ручне встановлення

1. Завантажте файли модів (`.jar` або `.zip`)
2. Помістіть їх у папку `./server/mods/`
3. Налаштуйте сервер для завантаження модів

## Увімкнення модів

Додайте аргумент `--mods` до конфігурації:

```bash
# .env
SERVER_EXTRA_ARGS=--mods mods
```

Або вкажіть власну папку модів:

```bash
SERVER_EXTRA_ARGS=--mods custom-mods-folder
```

## Власна папка модів

Ви можете підключити власну папку модів в `docker-compose.yml`:

```yaml
services:
  hytale-server:
    volumes:
      - ./server:/opt/hytale
      - ./my-mods:/opt/hytale/mods
```

## Інтеграція з Modtale

[Modtale](https://modtale.com) - це репозиторій модів для Hytale.

### Налаштування

1. Отримайте API ключ від Modtale
2. Додайте його до `.env`:

```bash
MODTALE_API_KEY=ваш-api-ключ
```

3. Перезапустіть панель:

```bash
docker compose restart hytale-panel
```

### Можливості

З інтеграцією Modtale:

- Перегляд модів прямо в панелі
- Перегляд описів, авторів та версій
- Встановлення та оновлення одним кліком
- Автоматичне вирішення залежностей

## Команди керування модами

З консолі сервера:

```
/mods list          # Список встановлених модів
/mods reload        # Перезавантажити всі моди
/mods enable <mod>  # Увімкнути мод
/mods disable <mod> # Вимкнути мод
```

## Усунення несправностей

### Моди не завантажуються

1. Перевірте, що `SERVER_EXTRA_ARGS` включає `--mods mods`
2. Переконайтеся, що моди в правильній папці
3. Перевірте логи сервера на помилки

### Проблеми сумісності

- Перевірте, що версія моду відповідає версії сервера
- Перевірте вимоги залежностей
- Спробуйте вимкнути інші моди для ізоляції проблеми

### Проблеми продуктивності з модами

- Моніторте використання RAM сервером
- Збільште `JAVA_XMX` за потреби
- Деякі моди можуть вимагати більше ресурсів
